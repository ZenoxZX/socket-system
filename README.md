# Socket System

A Unity package for rig socket management — define named sockets on a rig's bones for attaching objects (weapons, props), IK targets, or measurement points.

The package lives under [`src/Packages/com.zenoxzx.socketsystem`](src/Packages/com.zenoxzx.socketsystem). See its [README](src/Packages/com.zenoxzx.socketsystem/README.md) for features and usage.

## Install (Package Manager — Git URL)

**Window → Package Manager → + → Add package from git URL...**

```
https://github.com/ZenoxZX/socket-system.git?path=src/Packages/com.zenoxzx.socketsystem
```
